# PRODUCT.md

## Register
**Product** — app UI, dashboard, tool. Design SERVES the product.

## Users & Purpose
- **使用者**: 个人
- **场景**: 日常消费记账，桌面浏览器操作
- **核心任务**: 快速存入消费 → 多维度查询消费数据 → 了解消费趋势

## Brand Personality
现代、精致、可靠 — 个人工具但要有专业感

## Anti-references
- 避免过于花哨的装饰性动画
- 避免沉闷的企业级 ERP 风格
- 避免杂乱的信息堆砌

## Accessibility
- 桌面端优先
- 支持浅色/深色双主题
- 需要足够的颜色对比度

## Tech Stack
Java 17+, JDK HttpServer, MySQL, HTML/CSS 内联渲染
